# ARA Lite 0.2 / ARA Pro 2.2 - Frontend

Frontend modular y transparente para el framework de orquestación de modelos educativos y profesionales.

## Componentes principales

- **BudgetDashboard**: Transparencia de presupuesto y uso de tokens/modelos.
- **ModelSelector**: Selección de modelo local/cloud.
- **HumanGate**: Intervención humana en el ciclo de orquestación.
- **PromptPanel**: Envío de prompts y configuración de parámetros.
- **ResultViewer**: Visualización de resultados generados.
- **TelemetryPanel**: Monitoreo de métricas y telemetría.
- **RouterMetricsPanel**: Métricas del orquestador y fallback.
- **EmergencyStop**: Parada de emergencia y control inmediato.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Compilación

```bash
npm run build
```

## Previsualización

```bash
npm run preview
```
